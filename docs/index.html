Dikto

Voice-to-text for macOS. Records speech, transcribes locally. No cloud.

macOS 14+ Apple Silicon MIT License

Install

One-liner Recommended

Downloads, verifies, installs, and bypasses Gatekeeper automatically.

curl -fsSL https://diktoapp.github.io/dikto/install.sh | bash

Homebrew

Install the GUI app via Homebrew Cask.

brew install diktoapp/dikto/dikto --cask

Download DMG

Grab the latest DMG from GitHub Releases.

github.com/diktoapp/dikto/releases/latest

CLI only

Install just the command-line tool (builds from source).

brew install diktoapp/dikto/dikto

Features

Local transcription

Whisper and NVIDIA Parakeet models run entirely on your Mac. Nothing leaves your machine.

One shortcut

Press Option+R to record. Text is transcribed, copied to clipboard, and pasted into the active app.

Multiple models

From 75 MB (Whisper Tiny, 99 languages) to 2.6 GB (Parakeet v3, 25 EU languages). Pick your tradeoff.

Menu bar app

Lives in the menu bar. SwiftUI native. Minimal footprint. Launches at login.

Your voice stays on your device

Dikto never connects to a server. All speech processing happens locally using on-device ML models. No accounts, no telemetry, no cloud APIs.

Gatekeeper: "Dikto can't be opened" (DMG downloads only)

Dikto is ad-hoc signed but not Apple-notarized. If you download the DMG directly, macOS may block the app. The Homebrew and install-script methods bypass this automatically. For DMG installs, use one of these options:

  1. Right-click > Open: Right-click (or Control-click) Dikto.app, choose "Open", then click "Open" in the dialog.
  2. Terminal: Run xattr -cr /Applications/Dikto.app then open the app normally.
  3. System Settings: Go to System Settings > Privacy & Security, scroll down, and click "Open Anyway" next to the Dikto message.
